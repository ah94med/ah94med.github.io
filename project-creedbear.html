Creed & Bear Website Redesign

Case Study . Redesign . UI Design

Project Overview

Creed and Bear AG, is a young and dynamic company, that deals with the research and development of fintech technological solutions, applicable mainly to the financial sector. The Redesign was a skills assessment test, The aim was identify weakness in the company website and redesign it based on a strategy and the proper UX design process

Role

UI & UX Design

Discovery Phase

the discovery phase included, a UX audit of the website in terms of usability heuristics, a visual design audit, and a competitive analysis.

Defining Phase

the defining phase included, a user persona to guide the design decisions, a user journey map to define the user potential interaction with the website.

Ideation Phase

the ideation phase included, card sorting to map out ideas and content, user flow to map out the intended user journey on the website.

Design Phase

After the strategy and user flow, exploration of the design was done using pen and paper followed by high-fidelity design.
